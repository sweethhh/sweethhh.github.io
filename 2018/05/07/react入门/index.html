React入门笔记

React
  • 框架

重新学习React。

特性

虚拟DOM，在每次更新时与之前DOM对比，只更新我们改变的DOM。组件化，组件横向划分。

与vue不同
vue是双向数据绑定，react是单向，从父节点到子节点。

生命周期

三种状态

  1. mounting:正在挂载虚拟DOM到真实DOM
  2. updating:正在被重新渲染
  3. unmounting:正在将虚拟DOM移除真实DOM

三种状态对应的钩子

  1. componentWillMount() 即将要挂载
  2. componentDidMount() 挂载完成
  3. componentWillUpdate(object nextProps, object nextState) 将要更新
  4. componentDidUpdate(object prevProps, object prevState) 已更新
  5. componentWillUnmount() 将要卸载
